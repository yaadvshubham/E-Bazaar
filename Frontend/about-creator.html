Engineering Journal & Solved Challenges

Asynchronous Seeding & PostgreSQL Bloat Control
Resolved
Hurdle: When populating catalogs via seeder scripts or bulk scraper feeds, basic bulk insertion scripts crashed due to Supabase timeouts or database transaction memory bounds, and cleared user order ledgers unnecessarily.
Solution: Built an additive seeder wrapper that maps brand directories recursively. It implements batch insert buffers (100 rows per batch) and performs pre-insertion select queries to skip duplicate titles, protecting active order/user records.
Optimistic UI Caching & Timing Overlays
Resolved
Hurdle: Frequent network queries to Supabase resulted in brief flashes of raw pages, lowering E-Bazaar's premium aesthetic. But showing spinners on instant category shifts blocked the viewport and degraded UX.
Solution: Coded a client-side Stale-While-Revalidate (SWR) cache in `api.js` utilizing `localStorage`. Added a 300ms conditional delay threshold on loader overlays. Navigational or cached clicks render instantly with zero loader displays, while slow fetches (>300ms) fade in the brand overlay smoothly.
No-Transitions Theme & State Synchronization
Resolved
Hurdle: Navigating between catalog pages in Dark Mode caused bright-white blinks during initial rendering (before script loads), and elements re-animated their theme layout shift.
Solution: Injected self-executing inline theme-application scripts inside the `` of all pages to set themes prior to body parsing. Created a `.no-transitions` style block to disable animations during page load, which is removed via a 120ms timeout on script readiness.
